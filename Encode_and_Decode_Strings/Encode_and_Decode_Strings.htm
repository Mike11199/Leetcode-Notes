     Approach # 1 (Best) - Use a Delimiter and Add Integers to Track Word Length

 

 

Time Complexity
    •  O(N)

     

     

Space Complexity

    •  O(N)

     

     

Summary
    • ENCODE
      • Loop through the list of strings and append the length, '#' char, and string to a new string to later be decoded. 

     

    • DECODE
      • On the first loop [ j ] is set to [ i ] and [ j ] increments until it finds a '#' char.  This finds the string len that was encoded.

     

    • Then it converts the string number before '#'  char to an int

     

    • The int is used to read the num of chars of the string and append that slice back to an array of strings.

     

    • Then the  [ i ] variable is advanced past the string [ i + j + length ] to start reading what the next number int will be.
