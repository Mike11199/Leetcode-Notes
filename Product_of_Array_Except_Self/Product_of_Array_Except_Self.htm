     Approach # 1 (Best) - O(1) Constant Space Approach

 

 

 

Time Complexity
    •  O(N)

     

    • Have to iterate through numbers twice, once for the prefixes, another for postfixes.

     

Space Complexity

    •  O(1)

     

    • Answer array does not count, so we do all our work in the answer array.

     

Summary
    • Create array called result the same len as nums, populated with all 1s.  Only use this array going forward.

     

    • We want to continually track the prefix/postfix values with a variable that fills the result array then updates before incrementing

     

    • CREATE PREFIX ARRAY
      • Initialize prefix to 1 and Assign prefix to result[i]
      • Multiply prefix by nums[i]
      • Iterate to next spot of array (from beg to end)

     

    • CREATE POSTFIX ARRAY
      • Initialize postfix to 1 and Multiply result[i] by postfix
      • Multiply postfix by nums[i]
      • move to next elem of array (from end to beg)

     

 

 

 

 

 

 

    • Remember it stops BEFORE this number.
    • So if we put 0 here, it stops at 1
    • If we put -1 here, it stops at 0, so it loops through all numbers

 

PREFIX LOOP

 

 

 

    • Assign prefix to result[i]
    • Multiply prefix by number to store past multiplicands
    • Iterate to next spot of array (from beg to end)

 

POSTFIX LOOP

 

 

 

    • Multiply result[i] by postfix
    • Multiply postfix by number store past multiplicands
    • move to next elem of array (from end to beg)

 

 

 

     Approach # 1 (Okay) - Left and Right Product Lists

 

 

 

Time Complexity
    •  O(N)

     

    • N = Number of elements in array . 
    • One iteration to construct L array, another for R array, and one more for answer array.

Space Complexity

    •  O(N)

     

    • We need to create two intermediate arrays to track products to the left and right of each index.

     

Summary
    • Initialize three arrays of len(nums) and populate all elements to 0.

     

    • L Array
      • Loop through nums starting at pos[1] and multiply nums[i-1] * L[i-1].   L[0]=1
    • R Array
      • Loop through nums BACKWARDS starting at pos[-1], and multiply nums[i+1] *R[i+1].   R[-1] = 1
    • Answer Array
      • Populate array by assigning each element to L[i] *R[i]

     

 

 

 

 

    • Answer Array = L[i] * R[i]
