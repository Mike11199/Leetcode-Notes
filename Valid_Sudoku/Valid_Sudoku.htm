     Approach # 1 (Best) - Use Sets For Rows/ Columns/ 3x3 Squares to Check for Duplicates

 

 

Time Complexity
    •  O( 92 )

     

    • Adding elements to a hash set is O(1)
    • Checking for duplicates in a hash set is O(1)

     

Space Complexity

    •  O( 92 )

     

    • We have to add dictionaries for the entire board space which is a 9 by 9 grid.

     

Summary
    • Use three dictionaries, where value is a list or set of nums, and key is the row/column, or 3x3 box.

     

    • Check rows for duplicates with a set/list.

     

    • Check columns for duplicates with a set/list.

     

    • Create a hashmap with the key being (row/3, column/3) and value being the set of the values in each 3x3 box to check for duplicates.

     

    • Return true if no duplicates, or false if duplicates

     

 

 

 

 

    • Can use another set of indices to track the 3x3 square boxes.

 

 

 

 

 

 

 

 

    • Take coordinates 4,4
    • Divide (integer division by 3)
    • Results in 1,1 or the correct 3 by 3 box

 

 

 

 

KEY

One hash set with row/column/3

VALUE 

Set to check if duplicates in 3x3 box
